Wat is Linux?
Linux is een gratis computerbesturingssysteem dat door een gemeenschap van open-source-ontwikkelaars wordt gemaakt. Linux dient als alternatief voor het bekendere Windows of OS/X. In tegenstelling tot deze mainstream besturingssystemen is Linux een crowd-sourced project dat iedereen kan veranderen. Technisch gezien is Linux een "kernel", oftewel het fundamentele onderdeel van het besturingssysteem. Het systeem kan ook worden ge√Ønstalleerd om naast Windows te draaien en de schijf te delen. Terwijl andere besturingssystemen hun code beschermen tegen inmenging van buitenaf, verwelkomt Linux samenwerking. Deze geest heeft Linux een reputatie opgeleverd als broedplaats voor innovatie. Linux heeft functies ontwikkeld die uiteindelijk door grote besturingssystemen werden overgenomen.
Linux is gebaseerd op het vroegere Unix-systeem, dat het principe van modulair ontwerp gebruikte. In dit type systeem biedt het OS eenvoudige hulpmiddelen, terwijl shell scripting en commandotaal worden gecombineerd om complexe werkprocessen uit te voeren. Linux ondersteunt een aantal talen, waaronder PHP, Perl, Ruby en Python. Deze diversiteit geeft ontwikkelaars de vrijheid om code te schrijven in de taal waar ze zich het prettigst bij voelen.
